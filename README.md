# Portfolio Personal | Joaquín Seitz

## 📄 Descripción del Proyecto
Este repositorio contiene el código fuente de mi portfolio web personal, desarrollado como Trabajo Práctico Integrador para la **Tecnicatura Superior en Desarrollo de Software y Robótica**. 

El objetivo principal de este sitio es presentar mi perfil profesional de manera estructurada, destacando mi fuerte interés y enfoque en el desarrollo web y JavaScript, junto con mi experiencia práctica resolviendo problemas de hardware y software en entornos técnicos. El diseño detalla mi trayectoria académica, experiencia laboral, vías de contacto directo y proyectos destacados (como el proyecto integrador *GloveTec*), priorizando la usabilidad mediante una interfaz minimalista, moderna y de fácil navegación.

## 🛠️ Tecnologías Utilizadas
El sitio fue maquetado desde cero siguiendo estándares actuales de desarrollo Front-End:

* **HTML5:** Estructura semántica dividida modularmente en múltiples páginas (`index.html`, `experiencias.html`, `proyectos.html`, `contacto.html`).
* **CSS3:** Estilos en cascada personalizados, variables globales (`:root`) para mantener la coherencia de la paleta de colores corporativa, animaciones de transición fluidas (efectos hover) y la implementación de la fuente tipográfica personalizada *Modeska*.
* **Bootstrap 5.3:** Framework CSS principal utilizado para garantizar un diseño 100% responsivo (Mobile-First) mediante su sistema de grillas (Grid System). Se implementaron componentes nativos como Navbars fijas, Accordions (para el historial laboral) y Cards (para la exhibición de proyectos).
* **Bootstrap Icons:** Sistema de iconografía vectorial utilizado para optimizar la interfaz de usuario en las secciones de redes sociales y contacto.

El proyecto es un sitio web estático, por lo que no requiere la instalación de un servidor local ni dependencias de bases de datos. Para visualizarlo en cualquier entorno local, sigue estos pasos:

1. **Clonar el repositorio:**
   Abre tu terminal o consola de comandos y ejecuta:
   ```bash
   git clone [https://github.com/JoaquinSeitz/portfolio-joaquin-seitz.git](https://github.com/JoaquinSeitz/portfolio-joaquin-seitz.git)
Acceder al directorio:
Si descargaste el archivo ZIP desde GitHub, descomprímelo. Luego, ingresa a la carpeta raíz del proyecto clonado.

Ejecutar la aplicación:
Haz doble clic sobre el archivo index.html. El sitio se abrirá automáticamente en tu navegador web predeterminado y la navegación entre las distintas vistas funcionará de manera nativa.
